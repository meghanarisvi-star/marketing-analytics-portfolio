# Meta Ads ROAS Performance Dashboard

## Overview
This project analyzes Meta (Facebook & Instagram) advertising campaigns to evaluate performance using key marketing metrics such as ROAS (Return on Ad Spend), CTR (Click-Through Rate), conversions, and revenue.

The goal is to identify winning and losing ad campaigns and provide data-driven recommendations to optimize ad spend and improve profitability.

## Business Problem
Companies running paid ads often struggle with:
- Identifying profitable ads
- Understanding wasted ad spend
- Scaling high-performing campaigns
- Improving return on investment (ROI)

This project solves this by analyzing ad performance data and building a visual dashboard.

## Dataset Description
The dataset contains 20 Meta ad campaigns with the following columns:

- Ad Name 
- Spend  
- Impressions  
- Clicks  
- CPC  
- Conversions  
- Revenue  
- CTR (Clicks ÷ Impressions)  
- ROAS (Revenue ÷ Spend)

## Key Metrics
- CTR (Click-Through Rate) = Clicks / Impressions  
- ROAS (Return on Ad Spend) = Revenue / Spend  

## Tools Used
- Google Sheets → Data cleaning & calculations  
- Looker Studio → Dashboard creation  
- Data visualization → Charts & KPI cards  

## Key Insights
- Retargeting ads generated the highest ROAS
- Video-based ads outperformed static creatives
- Cold audience campaigns showed low profitability
- High CTR strongly correlated with higher conversions

## Business Recommendations
- Scale high-performing retargeting campaigns
- Pause or optimize low ROAS ads
- Improve creative hooks and storytelling
- Focus on video and UGC-style ads
- Test better audience segmentation
