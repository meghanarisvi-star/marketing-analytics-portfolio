# CRM Lead Pipeline Analysis & Sales Performance Dashboard

## Project Overview

This project analyzes a simulated CRM lead pipeline for a skincare ecommerce company. The objective was to evaluate lead sources, monitor sales pipeline performance, identify conversion opportunities, and present business insights through an interactive dashboard.

## Business Objective

* Analyze lead acquisition channels
* Evaluate CRM pipeline performance
* Monitor sales pipeline value
* Identify opportunities to improve lead conversion
* Build an executive dashboard for business reporting

## Dataset

A simulated CRM dataset containing 300 leads was created using AI. The dataset includes:

* Lead ID
* Lead Source
* Pipeline Status
* Sales Owner
* Days in Pipeline
* Conversion Status
* Lead Value

## KPIs Measured

* Total Leads
* Converted Leads
* Conversion Rate
* Total Pipeline Value
* Average Lead Value
* Average Days in Pipeline

## Business Insights

* Instagram generated the highest number of leads (120), making it the strongest acquisition channel.
* Google Ads produced nearly the same pipeline value despite fewer leads, indicating strong lead quality.
* Most leads remain in the early pipeline stages, highlighting opportunities for lead nurturing.
* Facebook contributed the fewest leads and the lowest pipeline value.

## Recommendations

* Continue investing in Instagram for lead generation.
* Optimize Google Ads campaigns to increase lead volume while maintaining lead quality.
* Improve follow-up strategies to increase conversions from Qualified and Proposal Sent stages.
* Review Facebook campaign targeting and budget allocation.
